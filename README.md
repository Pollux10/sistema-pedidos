# Sistema de Pedidos (CLI)

Sistema simples de gerenciamento de pedidos via terminal, desenvolvido em Python.

## 🚀 Funcionalidades

- Cadastro de produtos
- Listagem de produtos
- Realização de pedidos
- Listagem de pedidos
- Persistência de dados em JSON

## 🛠 Tecnologias

- Python
- JSON

## ▶️ Como executar

```bash
python main.py
